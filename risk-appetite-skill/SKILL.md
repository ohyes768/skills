# 风险偏好分析 Skill

## 触发条件

当用户需要分析**A股市场风险偏好**时触发此 Skill。

**具体场景：**
- 询问"当前市场风险偏好如何"
- 询问"近期市场情绪怎么样"
- 询问"杠杆资金动向"或"融资余额变化"
- 询问"两市成交额"或"换手率"
- 需要了解市场短期热度的场景

**本 Skill 不处理：** 个股分析、债券市场、汇率、CPI/PPI等宏观经济数据

## 数据获取流程

### 第一步：数据抓取

运行 `scripts/run_all.py` 获取所有风险偏好指标数据：

```bash
# 在 skill 目录下运行
uv run python scripts/run_all.py --days 5 --output ./risk_data.json
```

参数说明：
- `--days N`: 评估区间天数，默认5日
- `--output`: 输出 JSON 数据文件路径
- `--report`: 输出文本报告路径

### 数据抓取优先级

脚本内部有**两层 fallback 机制**：

1. **成交额/换手率**：`fetch_volume_exchange.py`（沪深交易所官方API）→ `fetch_volume.py`（akshare 指数数据 fallback）
2. **融资融券**：`fetch_margin.py`（akshare 中证数据）

具体实现：
- `fetch_volume_exchange.py` 直接调用沪深交易所官方 API，是**优先数据源**
- `fetch_volume.py` 在官方 API 失败时使用 akshare stock_zh_a_hist（上证指数+深证成指）作为 fallback
- `fetch_margin.py` 使用 akshare macro_china_market_margin_sh/sz（来自中证数据公司）

### 第二步：数据交叉确认

脚本获取的数据可能存在误差，引导用户核对：

---

**请核对以下数据是否与官方来源一致：**

| 指标 | 脚本获取值 | 官方数据来源 |
|------|-----------|-------------|
| 两市成交额 | {total_amount}亿元 | 沪深交易所每日行情 |
| 上证换手率 | {turnover_rate}% | Wind/东方财富 |
| 融资余额 | {rzye}亿元 | 中证数据有限责任公司 |

**如果数据不一致，请提供正确数值。**

## 分析框架

### 指标一：交易活跃度（权重50%）

由**成交额**（25%）和**换手率**（25%）两个子指标构成。

#### 子指标A：成交额（亿元）

| 成交额（单日） | 成交额（5日均值） | 市场状态 | 得分区间 |
|--------------|------------------|---------|---------|
| > 2.5万亿 | > 2.0万亿 | 极度亢奋 | 80–100 |
| 1.8–2.5万亿 | 1.3–2.0万亿 | 偏热 | 60–80 |
| 0.8–1.8万亿 | 0.8–1.3万亿 | 中性 | 40–60 |
| 0.5–0.8万亿 | 0.5–0.8万亿 | 偏冷 | 20–40 |
| < 0.5万亿 | < 0.5万亿 | 极度恐慌 | 0–20 |

#### 子指标B：换手率

| 单日换手率 | 5日均值 | 市场状态 | 得分区间 |
|-----------|--------|--------|---------|
| > 2.5% | > 2.0% | 极度亢奋 | 80–100 |
| 1.5%–2.5% | 1.2%–2.0% | 偏热 | 60–80 |
| 0.7%–1.5% | 0.7%–1.2% | 中性 | 40–60 |
| 0.4%–0.7% | 0.4%–0.7% | 偏冷 | 20–40 |
| < 0.4% | < 0.4% | 极度恐慌 | 0–20 |

**活跃度维度得分** = 成交额得分 × 50% + 换手率得分 × 50%

> 注：若单日值与均值信号指向矛盾，以均值为准（过滤短时脉冲噪音）。

### 指标二：融资融券余额（权重50%）

主要依据**融资余额环比变化率**（最新 vs 前一交易日）。

| 融资余额环比变化 | 市场状态 | 得分区间 |
|----------------|--------|---------|
| 单日 +2.5%以上，或5日累计 +5%以上 | 极度亢奋 | 80–100 |
| 单日 +1%~+2.5%，或5日累计 +2%~+5% | 偏热 | 60–80 |
| 单日 -1%~+1% | 中性 | 40–60 |
| 单日 -2.5%~-1%，或5日累计 -5%~-2% | 偏冷 | 20–40 |
| 单日 -2.5%以上，或5日累计 -5%以上 | 极度恐慌 | 0–20 |

**辅助加减分：**
- 融资余额连续5个交易日净流入：+5分
- 融资余额连续3个交易日净流入：+2分（介于两者之间）
- 融资余额连续3个交易日净流出且呈加速趋势：-5分
- 融资余额连续2个交易日净流出：-2分
- 融券余额出现异常跳升（单日环比翻倍等）：扣3–5分

### 综合评分与判定

```
加权总分 = 活跃度维度得分 × 50% + 融资融券得分 × 50%
```

| 总分 | 结论 | 操作建议 |
|------|------|---------|
| ≥80分 | 极度亢奋 | 警惕短期过热风险，不宜追高，注意仓位管理 |
| 60–79分 | 偏热/乐观 | 趋势向好，可参与但保持警惕 |
| 40–59分 | 中性 | 多空均衡，正常观察即可 |
| 20–39分 | 偏冷/谨慎 | 情绪低迷，可能是逐步低吸的机会 |
| <20分 | 极度恐慌 | 市场恐慌，通常是中长期左侧布局窗口 |

### 指标分歧处理

1. **若成交热度偏高但融资滞涨或流出**：优先信任融资数据，因杠杆资金更前瞻，属"聪明钱"信号
2. **若融资快速流入但成交不配合**：关注是否出现诱多陷阱，需结合指数走势判断
3. **单日异常值处理**：若某日因重大事件造成脉冲式波动，应参考近5日均值来平滑判断

## 注意事项

1. **成交额数据**：使用上证指数+深证成指分别代表沪市和深市，合计为两市成交额
2. **换手率数据**：使用上证指数换手率作为全市场参考指标
3. **融资融券数据**：每日09:45左右更新前一交易日数据，存在1天延迟
4. **指标权重**：活跃度50%（成交额25%+换手率25%），融资融券50%
5. **信号优先级**：当成交与融资指标出现分歧时，优先信任融资数据
6. **极值处理**：综合得分≥80或<20时，提示反转风险
