# Windows Batch Script Development

编写 Windows 批处理脚本（.bat/.cmd）的最佳实践和常见陷阱。

## 使用场景

当你需要：
- 编写 Windows 批处理脚本（.bat/.cmd）
- 创建安装/运行脚本
- 设置项目自动化脚本

## 技能内容

- ✅ 编码问题解决（避免中文乱码）
- ✅ 路径处理（从脚本位置定位）
- ✅ 虚拟环境处理（直接调用 Python）
- ✅ 错误检查和处理
- ✅ 脚本模板

## 核心原则

1. **全英文** - 避免编码问题
2. **cd /d "%~dp0.."** - 从脚本位置定位
3. **直接调用 python.exe** - 不依赖 activate
4. **检查文件存在** - 优雅处理错误
5. **pause 结束** - 让用户看到输出
