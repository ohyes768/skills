# Find Skills

帮助你发现和安装 Claude Code 技能生态系统中的其他技能。

## 使用方法

当你想要：
- 查找某个领域的技能（如 "find skills for React testing"）
- 询问是否有现成的技能可以完成某项任务
- 扩展 Claude Code 的功能

就可以使用这个技能。

## Skills CLI 命令

```bash
# 搜索技能
npx skills find [关键词]

# 安装技能
npx skills add <owner/repo@skill> -g -y

# 检查更新
npx skills check

# 更新所有技能
npx skills update
```

## 浏览技能

访问 https://skills.sh/ 查看所有可用技能。
