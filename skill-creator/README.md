# Skill Creator

帮助你创建、修改和优化 Claude Code 自定义技能。

## 使用方法

当你想要：
- 从零开始创建新技能
- 改进或优化现有技能
- 运行评估测试来验证技能性能
- 进行技能基准分析和性能对比
- 优化技能描述以提高触发准确性

就可以使用这个技能。

## 技能创建流程

1. **明确需求** - 确定技能的功能和实现方式
2. **编写草稿** - 创建技能的初始版本
3. **测试运行** - 使用测试提示词验证技能行为
4. **评估反馈** - 定性和定量分析技能表现
5. **迭代优化** - 根据反馈改进技能
6. **扩大测试** - 在更大规模测试集上验证

## 相关工具

- `eval-viewer/generate_review.py` - 查看评估结果
- `skills description improver` - 优化技能触发描述
